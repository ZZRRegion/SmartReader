# SmartReader
SmartReader是一款使用cefsharp+pdf.js作为核心架构开发的pdf阅读器，同时增加了google学术搜索模块，方便用户搜索相关文献；代理设置集成了另一款开源软件shadowsocket;
##SmartReader有哪些功能
* 本地文献库
  * 添加分组
  *  删除分组
  * 修改分组
  * 添加文献
 * 删除文献
  * 修改文献
  * 修改文献分组
* Google学术搜索
  * shadowsocket开源代理软件+比美特接口
* pdf文献阅读
  * 文献阅读进度
* 学习笔记
  * 添加截图笔记
  * 编辑截图笔记
